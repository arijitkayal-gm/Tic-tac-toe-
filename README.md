## 🚀 Features

- **Two-Player Mode:** Players X and O take turns.
- **Win Detection:** Alerts when a player wins.
- **Responsive Design:** Compatible with various screen sizes.
- **Restart Option:** Reset the game at any time.
- **Animated Background.
